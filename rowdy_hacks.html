Project Overview

Date: October 2024

This hackathon project was focused on developing a device tracking system using directional antennas and signal strength analysis.

My Contributions

  • Designed and constructed the antenna system with a stepper-motor driven sweep.
  • Integrated Arduino with Python for real-time movement and data collection.
  • Applied linear regression to estimate distances based on signal strength.
  • Used Kali Linux tools to locate and scan nearby network devices.

Skills Used

  • Python, Arduino, Serial Communication
  • Signal Strength Analysis & Regression
  • Kali Linux Networking Tools
